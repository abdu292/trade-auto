CAPITAL UTILIZATION — FORMULA BLOCK FOR RULES ENGINE
(Developer implementation block)

==================================================
INPUTS
==================================================

Cash_AED
Gold_Grams
MT5_Buy_Price_USD

Constants:
USD_TO_AED = 3.674
GRAMS_PER_OUNCE = 31.1035
SHOP_BUY_SPREAD = 0.80
CAPITAL_BUFFER = 0.995

==================================================
STEP 1 — REAL SHOP BUY PRICE
==================================================

ShopBuy_USD = MT5_Buy_Price_USD + SHOP_BUY_SPREAD

==================================================
STEP 2 — COST PER GRAM IN AED
==================================================

AED_Per_Ounce = ShopBuy_USD * USD_TO_AED

AED_Per_Gram = AED_Per_Ounce / GRAMS_PER_OUNCE

==================================================
STEP 3 — SAFE AVAILABLE CAPITAL
==================================================

AllowedCapital_AED = Cash_AED * CAPITAL_BUFFER

==================================================
STEP 4 — MAX LEGAL BUY GRAMS
==================================================

MaxLegalGrams = floor((AllowedCapital_AED / AED_Per_Gram) * 100) / 100

Note:
- floor to 2 decimals for precision safety
- if your execution supports only whole grams, then floor to integer grams instead

==================================================
STEP 5 — ORDER VALIDATION
==================================================

RequiredAED = OrderGrams * AED_Per_Gram

IF RequiredAED <= AllowedCapital_AED
    -> ORDER_STATUS = APPROVED

IF RequiredAED > AllowedCapital_AED
    -> ORDER_STATUS = RESIZE_REQUIRED
    -> ApprovedGrams = MaxLegalGrams

IF MaxLegalGrams <= 0
    -> ORDER_STATUS = REJECTED

==================================================
STEP 6 — IMMUTABLE EXECUTION GATE
==================================================

MT5 execution must only accept:

ApprovedOrder.approved_by_capacity_gate == true

Never send:
- raw AI output
- raw TABLE output
- raw candidate trade

Execution flow:

CandidateTrade
-> CapitalUtilizationCheck
-> ApprovedOrder OR RejectedOrder
-> MT5Execution

==================================================
STEP 7 — LOGGING
==================================================

Log these fields every time:

timestamp
cash_aed
mt5_buy_price_usd
shop_buy_price_usd
aed_per_gram
attempted_grams
max_legal_grams
required_aed
allowed_capital_aed
order_status
approved_grams

==================================================
EXAMPLE 1
==================================================

Cash_AED = 2237.42
MT5_Buy_Price_USD = 5171

ShopBuy_USD = 5171.80
AED_Per_Ounce = 5171.80 * 3.674 = 18997.7932
AED_Per_Gram = 18997.7932 / 31.1035 = 610.7924
AllowedCapital_AED = 2237.42 * 0.995 = 2226.2329

MaxLegalGrams = floor((2226.2329 / 610.7924) * 100) / 100
MaxLegalGrams = floor(3.6448 * 100) / 100
MaxLegalGrams = 3.64 g

==================================================

==================================================

MaxLegalGrams = floor((2226.2329 / 602.5110) * 100) / 100
MaxLegalGrams = floor(3.6957 * 100) / 100
MaxLegalGrams = 3.69 g
